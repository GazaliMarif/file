Biografi

Halo Nama saya Gazali Ma'rif, Saya lahir di Sungailiat, 5 Juni 2000. Saya merupakan anak 3 dari 3 bersaudara. Dan sekarang saya menetap di Sungailiat, Bangka Belitung. Saya juha sangat menyukai menonton dan bermain musik untuk menghilangkan stress. Saya adalah lulusan Politeknik Maunfaktur Negeri Bangka Belitung. Sewaktu kuliah saya mengambil jurusan Teknik Elektronika. Sekarang kesibukan saya agak berkurang tetapi saat ini saya melakukan pelatihan Website Programming untuk menambahkan skill yang saat ini saya miliki.
